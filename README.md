**A_DOGOVOR:**

* "ID_DOGOVOR" - идентификатор договора;
* "ID_USER" - идентификатор пользователя;

* "TRAFFIC" - безлимитный интернет трафик (Мбит/с);
* "DATA_CONECT" - дата подключения;
* "VALIDITY" - действителен до;
* "PRICE" - цена;
* "USER_LOGIN" - логин;
* "USER_PASSWORD" - пароль.

**A_USER:**

* "ID_USER" - идентификатор пользователя;
* "SURNAME" - фамилия;
* "NAME" - имя;
* "PATRONIMIC" - отчество;
* "PASPORT" - № паспорта;
* "ADRESS" - адрес;
* "TEL_NUMBER" - № телефона.
